# Support

## Getting Help

- For usage/setup questions: open a GitHub Discussion or Issue.
- For bugs: use the bug report template.
- For security topics: follow [SECURITY.md](SECURITY.md).

## What to Include

- HEXSONIC version/commit
- deployment mode (docker/systemd)
- logs (`docker compose logs -f api worker`)
- steps to reproduce

## Security Contact

For sensitive reports, do not post publicly.  
Use private maintainer contact or your internal security channel.
